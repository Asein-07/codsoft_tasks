# Rock Paper Scissors Game

## Description
This is a simple Rock Paper Scissors game developed using Python. The user plays against the computer, which randomly selects Rock, Paper, or Scissors. The program compares both choices, displays the result, and allows the user to play multiple rounds.

## Features
- User vs Computer gameplay
- Random computer choice
- Displays winner of each round
- Input validation
- Play multiple rounds
- Beginner-friendly Python code

## Technologies Used
- Python 3
- random module

## How to Run
1. Make sure Python 3 is installed.
2. Save the file as `rock_paper_scissors.py`.
3. Open a terminal or command prompt.
4. Navigate to the project folder.
5. Run the program using:

bash
python rock_paper_scissors.py
or
python3 rock_paper_scissors.py
