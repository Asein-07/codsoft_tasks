# Contact Book

## Description

This is a simple Contact Book application developed using Python.

The program allows users to store and manage contact information through a menu-driven interface.

## Features

- Add a new contact
- View all contacts
- Search a contact
- Update contact details
- Delete a contact
- Simple and easy-to-use interface

## Technologies Used

- Python 3

## How to Run

1. Download or clone the project.
2. Open the project folder.
3. Run the following command:

python main.py

## Example Menu

===== CONTACT BOOK =====

1. Add Contact
2. View Contacts
3. Search Contact
4. Update Contact
5. Delete Contact
6. Exit
