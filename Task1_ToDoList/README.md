# To-Do List using Python

## About the Project

This is a simple To-Do List application created using Python. The program runs in the command line and helps users organize their daily tasks. It allows users to add new tasks, view the existing task list, update tasks whenever needed, and remove completed tasks.

This project was developed to practice basic Python programming concepts such as lists, loops, conditional statements, and user input.


## Features

- Add a new task
- Display all tasks
- Edit an existing task
- Delete a task
- Simple menu-based interface
- Easy to understand and use


## Tools Used

- Python 3
- Visual Studio Code (or any Python IDE)

## How to Run

1. Download or clone this repository.
2. Open the project folder in your terminal.
3. Run the following command:

bash
python main.py


If your system uses Python 3 separately, run:

bash
python3 main.py
